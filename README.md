# InstaMart App

A full-stack grocery app built with Python.  
- **Backend:** Flask (server handles orders)  
- **Customer App:** Streamlit app to browse products and place orders  
- **Admin Dashboard:** Streamlit app to view orders in real-time  
- **Products:** Fruits, Dairy, Bakery, Groceries  
- **Port:** 5050 for server

## Features
- Browse products by category or search
- Add items to cart and checkout
- Orders appear in admin dashboard
